Руководство к созданной библиотеке функций, реализующих аппроксимации Паде.

	Содержание

1. Список реализованных функций
2. Формирование имен
3. Применение методов и задачи ими решаемые
4. Описание генерируемых программ


	Список реализованных функций

1. Ln(1 + x)
2. Arctg(x)
3. Arcsin(x)
4. Arcctg(x)
5. Arccos(x)


	Формирование имен

Для каждой реализованной функции сделано 2 класса по 3 и 2 метода соответственно.
Имя главного метода формируется следующим образом: (имя библиотеки) MathApprox + _ + <номер функции> + _ + 
<количество аргументов>. Во всех функциях количество аргументов 2 - это собственно аргумент функции
и допустимая погрешность.
Для удобства реализации создан также метод с именем, созданным из имени главного метода + _CheckError.
Имя третьего метода формируется добавлением _in к имени главного метода.

Второй класс содержит метод с именем, как главный метод, но с приставкой _Compute, и метод с приставкой _Compute_in.

Имена классов формируются следующим образом: C + MathApprox_ + <номер функции>

Так же есть отдельный класс CFind_Pade, содержащий две функции Find_Denominator и Find_Nominator, и класс CMakeProgram,
содержащий функцию MakeBeginning.


	Применение методов и задачи ими решаемые

Главный метод каждой функции принимает в качестве аргументов путь, в котором будет сгенерирована программа, и допустимую
погрешность для аппроксимации. Результат работы метода - программа, реализующая аппроксимацию заданной точности
в заданном каталоге.

Вспомогательный метод _CheckError принимает значения L и M - макисмальные степени числителя и знаменателя разложения.
На выходе имеем точность, с которой данная аппроксимация приближает функцию.

Метод второго класса _Compute получает на входе аргумент функции и допустимую погрешность. Результатом работы метода
является приближенное с заданной точностью значение функции.

Последний метод каждого из двух классов _in возвращает строку, содержащую информацию о области сходимости аппроксимации 
и возможных значениях погрешности.

Класс CFind_Pade призван вычислять коэффициенты разложения по известному разложению функции в ряд Тейлора и степеням
числителя и знаменателя. Первая функция вычисляет коэффициенты знаменателя, вторая - числителя (по уже известным
коэффицинетам знаменателя).

Последний класс CMakeProgram содержит функцию, которая по заданным пути, названию функции и погрешности создает файл
программы и записывает в него "шапку" программы. Возвращает значение типа System.IO.StreamWriter, чтобы затем можно было
продолжить запись в данный файл в главной функции.

Методы в консольном приложении MakeTests, MakeErrorPlots и MakeResFile соответсвенно тестируют заданную функцию, строят
графики по заданному файлу .csv и создают, собственно, отчетный файл .csv с абсолютной и относительной погрешностью.


	Описание генерируемых программ

Имя генерируемой программы формируется путем слияния имени главного метода и заданной погрешности вида aE-xx,
где а - мантисса, -хх - порядок (например, MathApprox_1_2_5E-10.cs). Сама программа содержит два массива Denom и Numer,
содержащие соответственно коэффициенты знаменателя и числителя аппроксимации (для заданной точности). Далее программа
выдает сообщение в консоли, сообщающее о том, что пользователь должен ввести значение аргумента для вычиления. После
считывания происходит вычисление значения функции с заданной точностью и вывод его на экран.
